## Critical Gaps

### 1. R2 Phase Transition Bug (RACE-01, RACE-04)

AppShell.tsx line 36 guards R2 transition behind `phase === 'group-stage'` but the store initializes at `'setup'` and no component transitions to `'group-stage'`. For 8+ team events, R2 is unreachable via the UI.

**Fix:** Change AppShell guard from `phase === 'group-stage'` to `(phase === 'group-stage' || phase === 'setup')`.

### 2. No Tiebreak Resolution UI (STND-03)

`setManualTiebreak` store action exists and is read by useR2State/useFinalsState, but no UI component calls it. When ties exist, the finals "blocked-ties" banner shows with no way for the official to resolve them.

**Fix:** Add a tiebreak resolution modal/interface that allows the official to drag-order tied teams.
