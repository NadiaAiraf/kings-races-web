    Hook that derives final placement results from scored finals matchups:
    ```typescript
    export function useFinalResults(discipline: DisciplineKey): FinalResult[] | null {
      const finalsState = useFinalsState(discipline);
      const { teams } = useDisciplineState(discipline);

      return useMemo(() => {
        if (!finalsState || finalsState.finalsPhase !== 'all-scored') return null;

        const teamMap = new Map(teams.map(t => [t.slot, t.name]));
        const results: FinalResult[] = [];

        // Process finals in reverse order (1st/2nd is last in array, highest placement)
        // Each finals matchup determines two positions from its label
        for (const matchup of finalsState.finalsWithNames) {
          if (!matchup.score || matchup.homeSlot === null || matchup.awaySlot === null) continue;

          // Parse label like "1st/2nd" -> positions [1, 2]
          const [winnerPos, loserPos] = parsePlacementLabel(matchup.label);

          const winnerSlot = matchup.score.homeOutcome === 'win' ? matchup.homeSlot : matchup.awaySlot;
          const loserSlot = matchup.score.homeOutcome === 'win' ? matchup.awaySlot : matchup.homeSlot;

          results.push({
            position: winnerPos,
            teamName: teamMap.get(winnerSlot) ?? `Team ${winnerSlot}`,
            placementLabel: formatOrdinal(winnerPos),
          });
          results.push({
            position: loserPos,
            teamName: teamMap.get(loserSlot) ?? `Team ${loserSlot}`,
            placementLabel: formatOrdinal(loserPos),
          });
        }

        // Sort by position ascending
        return results.sort((a, b) => a.position - b.position);
      }, [finalsState, teams]);
    }
    ```

    Also export a pure function `computeFinalResults(finalsMatchups, scores, teams, phase)` that returns `FinalResult[] | null`. This is the core logic extracted so it can be called without hooks -- both `useFinalResults` and the export handler use it. The function takes the resolved matchup data and score data directly, performs the same label parsing and winner/loser derivation, and returns sorted results. No hooks, no store access -- pure data in, data out.

    Helper: `parsePlacementLabel("1st/2nd")` -> [1, 2]. Parse the numeric values from the label string. Handle "3rd/4th", "5th/6th", "7th/8th", up to "31st/32nd".

    Helper: `formatOrdinal(n)` -> "1st", "2nd", "3rd", "4th", etc.

    Handle DSQ edge case: if one team DSQ'd, the other wins. The winner gets the higher placement, DSQ team gets lower. This is already handled by the score's homeOutcome/awayOutcome values.
