  <behavior>
    R2 seeding maps R1 group standings positions to R2 team slot numbers using the RoundTwoSeeding entries.

    resolveR2TeamSlot(seedingEntry, r1Standings, manualTiebreaks):
    - Input: a RoundTwoSeeding entry like { positionCode: 'A1', letter: 'A', label: 'Winner A' }
    - Parses label to extract position ("Winner"=1st, "Runner Up"=2nd, "Third"=3rd, "Loser"/"Last"=last) and group letter ("A", "B", "C", "D")
    - Looks up r1Standings[groupLetter][positionIndex].slot
    - If manualTiebreaks[groupLetter] exists, uses that ordering instead of standings order
    - Returns the slot number or null if unresolvable

    resolveR2GroupTeams(r2Group, r1Standings, manualTiebreaks):
    - Maps all seedingEntries in an R2 group to resolved slot numbers
    - Returns Record<string, number> mapping letter -> slot (e.g., { A: 1, B: 11, C: 2, D: 12 })
    - Returns null values for unresolvable entries

    areAllR1RacesScored(scores, structure):
    - Returns true when every race in structure.roundOneRaces has a score with raceId "r1-{raceNum}"

    areAllR2RacesScored(scores, structure):
    - Returns true when every R2 race across all R2 groups has a score with raceId "r2-{groupNum}-{raceNum}"
    - Returns true if structure has no roundTwoGroups (R1-only events)

    Test cases:
    - Test 1: "Winner A" with standings [slot 1 (9pts), slot 2 (6pts)] -> returns slot 1
    - Test 2: "Runner Up B" with standings [slot 11 (9pts), slot 12 (3pts)] -> returns slot 12 (2nd)
    - Test 3: "Third A" with 3-team group standings -> returns 3rd ranked slot
    - Test 4: "Loser A" with 4-team group -> returns last ranked slot
    - Test 5: With manualTiebreaks for group A, overrides natural standings order
    - Test 6: areAllR1RacesScored returns false when 1 race missing, true when all present
    - Test 7: areAllR2RacesScored returns true for R1-only structure (no roundTwoGroups)
    - Test 8: Full 8-team scenario: resolve all R2 Group I and Group II teams from R1 standings
  </behavior>

    resolveFinalsRef(ref, r2Standings, r1Standings, manualTiebreaks):
    - Parse ref string to extract position and group identifier
    - Determine whether to use R1 or R2 standings based on group identifier format
    - Return slot number or null

    All 32 unique ref patterns to handle:
    R1-only refs (Group A only):
    - "Winner Group A" -> 1st in Group A standings
    - "2nd Group A" -> 2nd in Group A standings
    - "3rd Group A" -> 3rd in Group A standings
    - "4th Group A" -> 4th in Group A standings
    - "5th Group A" -> 5th in Group A standings
    - "Loser Group A" -> last in Group A standings

    R2 refs (Group I, II, III, IV, V, VI):
    - "Winner Group {N}" -> 1st in R2 Group {N} standings
    - "Second Group {N}" -> 2nd in R2 Group {N} standings
    - "Third Group {N}" -> 3rd in R2 Group {N} standings
    - "Loser Group {N}" -> last in R2 Group {N} standings

    Anomalous refs (teams7 only):
    - "Winner I" -> 1st in R2 Group I standings
    - "Runner up I" -> 2nd in R2 Group I standings

    resolveAllFinalsMatchups(structure, r2Standings, r1Standings, manualTiebreaks):
    - Returns array of { label, homeRef, awayRef, homeSlot, awaySlot } for all finals

    areAllFinalsScored(finalsMatchups, scores):
    - Returns true when every matchup has a score with raceId "fin-{index}"

    Test cases:
    - Test 1: "Winner Group A" against R1 standings -> correct slot
    - Test 2: "2nd Group A" -> 2nd ranked slot
    - Test 3: "Loser Group A" -> last ranked slot
    - Test 4: "Winner Group I" against R2 standings -> correct slot
    - Test 5: "Second Group II" -> 2nd in R2 Group II
    - Test 6: "Winner I" (teams7 anomaly) -> 1st in R2 Group I
    - Test 7: "Runner up I" (teams7 anomaly) -> 2nd in R2 Group I
    - Test 8: Full 4-team event resolves both finals matchups correctly
    - Test 9: Full 8-team event resolves all 4 finals matchups correctly
    - Test 10: Exhaustive test -- iterate ALL 29 cheat sheets, verify resolveFinalsRef returns non-null for every homeRef/awayRef when given valid standings
    - Test 11: areAllFinalsScored returns true/false correctly
  </behavior>

    1. Try R1-only pattern: match /(Winner|2nd|3rd|4th|5th|Loser) Group ([A-Z])$/
    2. Try R2 pattern: match /(Winner|Second|Third|Loser) Group ([IVX]+)$/
