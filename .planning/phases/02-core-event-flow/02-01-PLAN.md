**useDisciplineState.ts** -- derives current discipline's full state:
- Import `useEventStore` and `getCheatSheet` from domain
- Accept `discipline: DisciplineKey` parameter
- Use Zustand selector: `useEventStore((s) => s.disciplines[discipline])` to get `{ teams, teamCount, scores, phase, manualTiebreaks }`
- Derive `structure` via `useMemo`: if `teamCount >= 4`, call `getCheatSheet(teamCount)`, else `null`
- Return `{ teams, teamCount, scores, phase, structure, manualTiebreaks }`
- Per Pitfall 5: use specific selector, NOT `useEventStore()` without selector

**useCurrentRace.ts** -- finds next unscored race for scoring focus:
- Accept `discipline: DisciplineKey` parameter
- Use `useDisciplineState(discipline)` to get `scores` and `structure`
- If no structure, return `{ currentRace: null, currentIndex: -1, totalRaces: 0, allRaces: [] }`
- Get all round-one races from `structure.roundOneRaces`
- Build a Set of scored raceIds from `scores.map(s => s.raceId)`
- Find first race where `r1-${race.raceNum}` is NOT in the scored set
- Return `{ currentRace, currentIndex, totalRaces: structure.roundOneRaces.length, allRaces: structure.roundOneRaces }`
- The race ID format is `r1-{raceNum}` for round 1 (per Pattern 4 from RESEARCH.md)

**useStandings.ts** -- reactive standings calculation:
- Accept `discipline: DisciplineKey` parameter
- Use `useDisciplineState(discipline)` to get `scores`, `teamCount`, `teams`, `structure`
- If no structure, return `null`
- Use `useMemo` with deps `[scores, teamCount, teams]`:
  - Call `calculateAllGroupStandings(scores, structure.groups)` from domain
  - Build `tiesByGroup: Record<string, boolean>` by calling `hasTies()` for each group
  - Return `{ standings, tiesByGroup, groups: structure.groups, teams }`
- Per Pitfall 2: always derive from current teamCount, never cache structure separately

**Toast.tsx** (per UI-SPEC Destructive Actions):
- Props: `message: string`, `onDismiss: () => void`, `duration?: number` (default 3000ms)
- Render at bottom of screen: `fixed bottom-4 left-1/2 -translate-x-1/2 z-50`
- Background: `bg-slate-900 text-white px-4 py-3 rounded-lg shadow-lg`
- Auto-dismiss after `duration` ms via `useEffect` with `setTimeout`

- Use `useState` for `confirming` boolean and `useEffect` with `setTimeout(3000)` to auto-revert
- Min height: `min-h-14` for touch target (56px per D-03, RESL-02)
