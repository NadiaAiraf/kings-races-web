## Interaction Contract

### Expandable Card Behavior

| Interaction | Behavior |
|-------------|----------|
| Tap unscored collapsed card | Expand to reveal Win/DSQ buttons. Collapse any previously expanded card. |
| Tap scored collapsed card | Re-expand to reveal Win/DSQ buttons pre-highlighted with current result (D-08). |
| Tap Win for a team | Record Win for that team, Loss for opponent. Collapse card. Auto-expand next unscored. |
| Tap DSQ for a team | Record DSQ for that team, Win for opponent. Collapse card. Auto-expand next unscored. |
| Tap outside expanded card | Collapse without changes. No score recorded. |
| Tap a different outcome on re-expanded scored card | Overwrite previous score with new result. Collapse card. |

### Auto-Expand (D-06)

- On initial Races tab load: expand the first unscored race in the active round.
- After scoring: collapse scored card, expand next unscored race in sequence.
- If all races in current round are scored: no card expanded, show round completion state.

### Auto-Scroll (D-07)

- After scoring and auto-expanding next card: smooth scroll to bring the newly expanded card into view.
- Use `scrollIntoView({ behavior: 'smooth', block: 'nearest' })` on the expanded card ref.
- Apply `scroll-margin-top: 56px` (to clear sticky sub-tabs at top-12 + h-11 = ~92px total from DisciplineTabs + SubTabs).

### Card Layout — Collapsed State

```
+--------------------------------------------------+
| Race 12              Group A                      |
| Team Alpha   [W]     vs     Team Beta   [L]      |
+--------------------------------------------------+
```

- Race number and optional group label on first line (text-sm text-slate-500).
- Left/right team layout: home team name left, "vs" center (text-slate-500), away team name right.
- Outcome badges (W/L/DSQ) appear inline next to team names when scored.
- Team names at text-base (16px) font-semibold text-slate-900.

### Card Layout — Expanded State

```
+--------------------------------------------------+
| Race 12              Group A                      |
|                                                   |
|  Team Alpha                    Team Beta          |
|  [  Win  ] [  DSQ  ]    [  Win  ] [  DSQ  ]      |
+--------------------------------------------------+
```

- Same header row as collapsed state.
- Two columns: home team left, away team right.
- Each column: team name (text-base font-semibold) above a row of two buttons (Win, DSQ).
- Buttons: h-14 (56px) height, flex-1 within their column, font-semibold text-base rounded-lg.
- Button gap: gap-2 (8px) between Win and DSQ within a team column.
- Column gap: gap-4 (16px) between home and away columns.
- Card padding: p-4 (16px) all sides.

### Scoring Buttons — Win/DSQ Only (D-04, D-05)

| Button | Tap Effect |
|--------|-----------|
| Win (home) | Records home=win, away=loss. Single tap resolves matchup. |
| DSQ (home) | Records home=dsq, away=win. Single tap resolves matchup. |
| Win (away) | Records away=win, home=loss. Single tap resolves matchup. |
| DSQ (away) | Records away=dsq, home=win. Single tap resolves matchup. |

No intermediate state. No Loss button. Every tap is a complete matchup resolution.

### Round Sections (D-10)

- Round 1, Round 2, Finals each have a RoundHeader divider.
- Active round: text-slate-900, cards fully interactive.
- Inactive future rounds: text-slate-400 opacity-40, cards show placeholder text.
- Inactive past rounds: text-slate-900 (normal color), cards show scored results, tappable for re-scoring.

### Finals in Races List (D-11)

- Finals matchups render as ExpandableRaceCard components under a "Finals" RoundHeader.
- Finals cards show matchup label (e.g., "1st/2nd Place") and seeding reference in the collapsed header area.
- Same Win/DSQ scoring pattern as group-stage cards.
- Unresolved finals matchups show "Teams not yet resolved" in text-slate-400, not expandable.
