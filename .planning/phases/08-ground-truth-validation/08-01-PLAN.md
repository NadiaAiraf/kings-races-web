<!-- The golden data JSON format per D-02 -->
Each entry keyed by team count string ("4" through "32"):
```json
{
  "teamCount": number,
  "seedMap": number[],       // seedMap[0] = slot for seed 1, seedMap[1] = slot for seed 2, etc.

  ],
  "r2Races": [               // R2 race matchups as flat sequence (empty array for 4-6 teams)

}
```

Create `scripts/extractGoldenData.py` that extracts golden data from `reference/cheat-sheets-v1.4.xlsx` and writes a single combined JSON fixture to `src/domain/cheatSheets/__fixtures__/goldenData.json`. Per D-01, use Python with openpyxl. Per D-02, output format is one combined JSON file keyed by team count string.

**Seed map extraction (per D-03):**
- Open workbook with `data_only=False` to read formulas
- For each sheet (4 Teams through 32 Teams), scan column C for formulas matching `='Enter Teams'!B{row}`
- Extract seed number as `row - 2` (row 3 = seed 1, row 4 = seed 2, etc.)
- Extract slot number from column B of the same row
- Sort by seed number, output as array where index i = slot for seed (i+1)
- **Per D-04:** For 32 Teams (no formulas in column C), hardcode the seedMap derived from 25-31 pattern: `[1,11,21,31,41,51,61,71,74,64,54,44,34,24,14,4,2,12,22,32,42,52,62,72,63,53,43,33,23,13,3,73]`
- Log a warning when 32 Teams uses the hardcoded fallback

**R1 race extraction:**
- Scan column J for patterns matching `N V M` (with optional trailing asterisk/spaces)
- Use regex `r'^(\d+)\s*V\s*(\d+)$'` on cleaned values (strip, rstrip `* `)
- Number races sequentially starting at 1
- Reuse patterns from `extractCheatSheets.py` for quirk handling

**R2 race extraction:**
- Scan column T for letter-based patterns matching `X V Y`
- Use regex `r'^([A-Za-z0-9]+)\s+V\s+([A-Za-z0-9]+)\s*$'` on cleaned values
- Handle `0` -> `O` substitution (known xlsx quirk per research pitfall 2)
- Normalize whitespace (strip, handle double spaces)
- For 4-6 teams, output empty `r2Races` array (no R2 round exists per pitfall 4)
- Extract as flat sequence -- do NOT group by R2 group (per research open question 1)

**Output:**
- Create `__fixtures__` directory if it does not exist
- Write JSON with `indent=2` for readability
- Print summary: team count, seedMap length, R1 race count, R2 race count for each sheet
- Validate: seedMap length should equal teamCount, log warning if mismatch
