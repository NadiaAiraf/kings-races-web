**Suite 1: VALID-01 -- Seed mapping validation**
```
describe('VALID-01: seed-to-slot mapping matches xlsx')
```
For each team count 4-32:
- Get `golden.seedMap` (array where index i = slot for seed i+1)
- Get `sheet = getCheatSheet(n)`
- Collect all slots from `sheet.groups.flatMap(g => g.teamSlots)`
- Verify every slot in `golden.seedMap` exists in the group slots set
- Verify total seedMap length equals `n` (teamCount)
- Use descriptive assertion messages: `"${n} teams: seed ${i+1} -> slot ${golden.seedMap[i]} not found in any group"`

**Suite 2: VALID-02 -- R1 race order validation**
```
describe('VALID-02: R1 race order matches xlsx')
```
For each team count 4-32:
- Get `golden.r1Races` and `sheet.roundOneRaces`
- Assert array lengths match: `"${n} teams: expected ${golden.r1Races.length} R1 races, got ${sheet.roundOneRaces.length}"`
- For each race, compare `homeSlot` and `awaySlot`:
  - `"${n} teams, race ${i+1}: expected home=${expected.homeSlot}, got ${actual.homeSlot}"`
  - `"${n} teams, race ${i+1}: expected away=${expected.awaySlot}, got ${actual.awaySlot}"`

**Suite 3: VALID-03 -- R2 race order validation**
```
describe('VALID-03: R2 race order matches xlsx')
```
For each team count 4-32:
- Get `golden.r2Races` (flat sequence)
- If `golden.r2Races` is empty (4-6 teams), assert `sheet.roundTwoGroups` is either undefined or empty, then skip
- For 7+ teams: flatten TS R2 races from `sheet.roundTwoGroups` into a single array by concatenating each group's `races` array in group order (Group I first, then II, etc.)
- Compare the flat sequences by matchup content (homeLetter, awayLetter) -- per research open question 1, compare flat sequence order, NOT per-group race numbers
- Use descriptive messages: `"${n} teams, R2 race ${i+1}: expected ${expected.homeLetter} V ${expected.awayLetter}, got ${actual.homeLetter} V ${actual.awayLetter}"`

**Important considerations:**
- Import golden data as: `import goldenData from './__fixtures__/goldenData.json'`
- Type the import: `const data = goldenData as Record<string, { teamCount: number; seedMap: number[]; r1Races: { raceNum: number; homeSlot: number; awaySlot: number }[]; r2Races: { raceNum: number; homeLetter: string; awayLetter: string }[] }>`
- Use a `for` loop with individual `it()` calls (not `it.each`) so each team count is a separate test that can fail independently
- Use `const teamCounts = Array.from({ length: 29 }, (_, i) => i + 4)` for the loop range
